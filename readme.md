# date-list-gen

> Generate a list of dates


## Install

```
$ npm install --global date-list-gen
```


## Usage

```
$ date-list-gen --help

  Usage
    $ date-list-gen [year] [dateFormat]

  Options
    year  Year to print. [Default: the current year]
    dateFormat  date format w/ Moment.js conventions. [Default: ddd MMM D]

  Examples
    $ date-list-gen

    $ date-list-gen 2016
```
